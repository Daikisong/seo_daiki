# Cross-Market Trend Map

The cross-market map classifies clusters as:

- local-only trend
- regional trend
- global trend
- lagging market opportunity
- cross-language synonym cluster

Example:

If USB-C chargers are trending in Spain and Mexico, that is regional. If the same cluster appears later in Brazil and Portugal, the map can show a lagging Portuguese-language opportunity.

The map is a summary layer. It does not merge market queues.
