# SERP Intelligence

SERP intelligence answers one question:

What are top-ranking pages doing, and what should we do differently?

Allowed provider modes:

- `manual_csv`
- `google_programmable_search_existing_customer`, disabled by default
- `third_party_serp_api`, disabled by default
- `browser_research_manual`
- `search_console_existing_pages`

Not allowed:

- raw Google HTML scraping
- proxy scraping
- CAPTCHA bypass
- automated rank-checking spam

The output is opportunity analysis, not product links.
