# SERP Provider Contract

Provider output must normalize into:

- `market`
- `language`
- `keyword`
- `rank`
- `url`
- `domain`
- `title`
- `snippet`
- `resultType`
- `dateHint`
- `isForum`
- `isVideo`
- `isEcommerce`
- `isAffiliateLikely`
- `isPublisher`

Providers must identify their collection mode and policy boundary.

Only `manual_csv` is enabled by default.
