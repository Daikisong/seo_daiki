# Feature Flags

Defaults:

```text
ENABLE_OFFER_MATCHING=false
ENABLE_DISTRIBUTION_DRAFTS=false
ENABLE_LINK_EARNING=false
ENABLE_LIVE_AFFILIATE_APIS=false
ENABLE_PRODUCT_CANDIDATE_DISCOVERY=true
ENABLE_SERP_INTELLIGENCE=true
ENABLE_TREND_ENGINE=true
```

Meaning:

- Offer matching is later phase.
- Distribution drafts are later phase.
- Link earning and outreach are later phase.
- Live affiliate APIs are documentation-only for now.
- Trend and SERP engines are on by default.
