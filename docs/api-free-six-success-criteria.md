# API-Free Six Success Criteria

The API-free step 1-6 flow passes only when every item below is true.

## Trend Outputs

- `data/exports/market_trend_sources.json` exists.
- `data/exports/market_trend_signals.json` exists.
- `data/exports/trend_clusters.json` exists.
- `data/exports/trend_keywords.json` exists.
- `data/exports/trend_report.json` exists.
- At least 5 market trend clusters exist.
- At least 5 trend keywords exist.
- At least 3 distinct markets exist in generated trend outputs.

## SERP Outputs

- `data/exports/serp_results.json` exists.
- `data/exports/competitor_content_analysis.json` exists.
- `data/exports/serp_opportunity_report.json` exists.
- At least 5 SERP results exist.
- At least 5 competitor analyses exist.
- At least 3 SERP opportunities exist.
- Each opportunity has `keywordId`, `market`, `language`, `keyword`, `dominantIntent`, `recommendedAngle`, `recommendedArticleType`, and `shouldWrite`.

## Strategy Outputs

- `data/exports/content_strategies.json` exists.
- `data/exports/content_briefs.json` exists.
- At least 3 content strategies exist.
- At least 3 content briefs exist.
- Each strategy has `keywordId`, `clusterId`, `market`, `language`, `selectedArticleType`, `recommendedAngle`, `titleStrategy`, `sectionPlanJson`, and `monetizationDeferred=true`.
- Each strategy is generated by the dry-run provider in API-free mode.

## Test Post Outputs

- `data/exports/test_articles.json` exists.
- At least 3 test articles exist.
- Each test article has `market`, `language`, `slug`, `title`, `h1`, `metaDescription`, `contentMdx` or `sections`, `affiliateLinks=[]`, `monetizationDeferred=true`, `productCandidateState=pending`, `indexStatus=noindex`, and `publishStatus=pending` or `published`.
- Test posts must not contain monetized affiliate links.

## Website Route Verification

- Each generated test article route is `/[market]/[language]/posts/[slug]/`.
- The web layer reads `data/exports/test_articles.json`.
- `generateStaticParams()` for posts includes generated posts.
- Post metadata is noindex through `marketResearchMetadata()` or equivalent.
- `pnpm build` passes.

## Safety

- No live affiliate APIs are called.
- No Search Console API is required.
- No external LLM API is required.
- No raw Google scraping is required.
- No product monetization is inserted.
- No distribution or outreach is run.

The executable check is:

```bash
pnpm verify:api-free-six
```
