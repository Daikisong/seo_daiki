# Anti-Spam Rules

The system intentionally avoids blackhat SEO automation.

Do not build or enable:

- comment spam
- forum/profile backlink creation
- account creation automation
- CAPTCHA bypass
- proxy rotation
- automated votes, likes, follows, or comments
- PBN or directory submission
- Google SERP scraping
- Google Indexing API submission for normal articles

Safe alternatives in this repo:

- sitemap-based crawling
- Search Console refresh suggestions
- owned-channel draft generation
- human-approved outreach drafts
- manual CSV imports
- official APIs only when explicitly configured

If a workflow would create a link or post without a human approval state, it does not belong in this project.
