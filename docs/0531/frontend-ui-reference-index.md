# 0531 Korean Frontend UI References

실제 상위 블로그/가이드/리뷰/공공정보 페이지에서 반복된 구조를 바탕으로 만든 한글 UI 레퍼런스다.

핵심 기준:

- 결론 또는 현재 상태를 먼저 보여준다.
- 업데이트 날짜, 검토 방식, 공식/1차 출처를 위쪽에 둔다.
- 긴 본문 전에 읽는 순서, 체크리스트, 비교표를 배치한다.
- 검색 분석 노트나 내부 워크플로 문구는 독자에게 보이지 않는다.
- 카드 밀도는 낮추고, 제목/본문/표의 위계를 명확히 한다.
